#biqh-angular-es5

## Getting Started

All you need to do is to clone this repository,

```
git clone https://github.com/mehmettugrulsahin/biqh-angular-es5
cd biqh-angular-es5
```

Then, install all the dependencies:

```
npm install
```

Then, run the Application:

```
npm start
```

Then, navigate to:
```
http://localhost:8080
```
